## 🚀 كيفية الاستخدام

### متطلبات المشروع
- متصفح ويب حديث (Chrome, Firefox, Safari, Edge)
- ملف PDF للمنيو (اختياري)

### الخطوات

1. **استنساخ المشروع:**
```bash
git clone https://github.com/Ali7usse1n/ali7usse1ncoffee.git
cd ali7usse1ncoffee
```

2. **فتح الموقع:**
   - افتح ملف `index.html` مباشرة في المتصفح
   - أو استخدم خادم محلي (مثل Live Server في VS Code)

3. **إضافة ملف المنيو:**
   - ضع ملف PDF باسم `menu.pdf` في نفس المجلد الأساسي
   - سيتم تحميله عند الضغط على زر "عرض المنيو الكامل"

## 🎮 الاختصارات لوحة المفاتيح

| الزر | الوظيفة |
|------|----------|
| `M` | فتح المنيو |

## 🔧 التخصيص

### تغيير الألوان
عدل متغيرات الألوان في `styles.css`:
```css
:root {
    --primary-brown: #8B4513;        /* اللون الأساسي البني */
    --accent-gold: #D4AF37;          /* اللون الذهبي */
    --light-bg: #f5e6d3;             /* لون الخلفية الفاتح */
}
```

### تغيير النصوص والصور
عدل الأجزاء التالية في `index.html`:
- عنوان الموقع
- وصف المقهى
- رابط الصور
- معلومات التواصل

### تحديث المنيو
استبدل ملف `menu.pdf` بملف المنيو الخاص بك
